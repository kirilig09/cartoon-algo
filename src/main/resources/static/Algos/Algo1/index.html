Bubble Sort

Statistics

Explanation

Bubble sort is an algorithm used to sort an array of elements. The algorithm repeatedly steps through the list, compares each pair of adjacent items and swaps them if the left one is bigger than the right. This way the algorithm guarantees that the biggest element always goes to the back.

Speed

The average speed is O(N ^ 2) and the best case is O(N), when the elements are sorted. One of the main advantages of this algorithms over the other ones is that it has the ability to check if the list is sorted. This means that it always does the minimum amount of actions it has to in order to sort the array.
